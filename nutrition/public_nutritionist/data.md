### 标准体重指数分级表

| 标准体重指数 | 评价 |
| -- | -- |
| >10% | 超重 |
| >=20% | 肥胖 |
| 20%~30% | 轻度肥胖 |
| 30%~50% | 中度肥胖 |
| >=50% | 肥胖 |
| >=100% | 病态肥胖 |

### 腰围数据

- 中国肥胖问题工作组建议腰围男性 >= 85 cm，女性 >= 80 cm 为肥胖的标准
- 国际糖尿病联盟的标准为男性腰围 >= 90，女性 >= 80
- 腹型肥胖的标准是：腰臀比（WHR）：男性 >= 0.9，女性 >= 0.85

> 记忆：腰围与腰臀比的关系：按臀围为 94 cm 来计算：85/94 = 0.90，80/94 = 0.84
